Why this exists

Most Python courses teach programming. Python for Mammals teaches useful automation through small, daily, beginner-friendly steps.

What you will build

By the end, you should be comfortable creating scripts for reports, file handling, logs, checks, data cleanup, and repeatable daily tasks.
