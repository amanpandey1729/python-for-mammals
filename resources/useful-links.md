# When To Use What?

| Situation                  | Recommended Resource      |
| -------------------------- | ------------------------- |
| Python installation issues | Official Python Website   |
| VS Code issues             | VS Code Documentation     |
| Understand how code works  | Python Tutor              |
| Quick online testing       | Replit                    |

# Important Advice

Do not try to learn from ten different websites at the same time.

For this challenge:

1. Follow the daily lessons.
2. Complete the exercises.
3. Use these resources only when needed.
4. Stay consistent.

Thirty minutes every day is better than five hours once a week.

Happy learning. 🐍
