# Python for Mammals Glossary

This glossary contains common terms you will encounter throughout the 45-Day Python Challenge.

Bookmark this page and revisit it whenever you encounter unfamiliar terminology.

---

# Algorithm

A step-by-step set of instructions used to solve a problem.

### Example

Making tea:

1. Boil water
2. Add tea
3. Add milk
4. Serve

This is an algorithm.

---

# Argument

A value passed to a function.

### Example

```python
print("Hello Mammals")
```

Here:

```python
"Hello Mammals"
```

is an argument passed to the `print()` function.

---

# Bug

A mistake or error in a program that causes unexpected behavior.

### Example

```python
print("Hello Mammal)
```

This contains a bug because a quotation mark is missing.

---

# Comment

Text written in code that Python ignores.

Used to explain code.

### Example

```python
# This is a comment
print("Hello")
```

---

# Data Type

The category of a value.

### Common Data Types

| Type    | Example  |
| ------- | -------- |
| String  | "Python" |
| Integer | 10       |
| Float   | 3.14     |
| Boolean | True     |

---

# Function

A reusable block of code that performs a task.

### Example

```python
print("Hello")
```

`print()` is a function.

---

# IDE

Integrated Development Environment.

A tool used to write and run code.

### Examples

* VS Code
* PyCharm

---

# Indentation

Leading spaces used to organize Python code.

Python uses indentation to define code blocks.

### Example

```python
if True:
    print("Hello")
```

---

# Interpreter

The software that reads and executes Python code.

When you run:

```bash
python myfile.py
```

the Python interpreter executes the file.

---

# Keyword

A reserved word that has a special meaning in Python.

### Examples

```python
if
else
for
while
def
return
```

---

# Library

A collection of modules that provide additional functionality.

### Examples

* pandas
* requests
* openpyxl

---

# Loop

A structure used to repeat actions.

### Example

```python
for i in range(3):
    print(i)
```

Output:

```text
0
1
2
```

---

# Module

A Python file containing reusable code.

### Example

```python
import math
```

Here:

```python
math
```

is a module.

---

# Package

A collection of related modules.

### Example

```python
pandas
```

is a package containing many modules.

---

# PATH

An operating system setting that helps locate programs.

If Python is added to PATH, you can run:

```bash
python --version
```

from any directory.

---

# Program

A set of instructions written to perform a task.

### Example

```python
print("Hello Mammals")
```

is a very small program.

---

# Script

A Python file containing code.

### Example

```text
hello.py
```

---

# String

Text enclosed within quotes.

### Example

```python
"Python"
```

```python
'Hello Mammals'
```

---

# Syntax

The rules of a programming language.

### Correct

```python
print("Hello")
```

### Incorrect

```python
print("Hello"
```

---

# Terminal

A text-based interface used to execute commands.

### Examples

* Command Prompt
* PowerShell
* Terminal (Linux/macOS)

---

# Variable

A named container used to store information.

### Example

```python
name = "Aman"
```

Here:

```python
name
```

is a variable.

---

# Version Control

A system that tracks changes to files over time.

### Example

Git

Benefits:

* Track changes
* Recover old versions
* Collaborate with others

---

# VS Code

Visual Studio Code.

The editor used throughout this challenge.

---

# Quick Advice

Do not try to memorize everything.

The purpose of this glossary is to provide a quick reference whenever you encounter an unfamiliar term.

As you practice daily, these terms will gradually become part of your vocabulary.
