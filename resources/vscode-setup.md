# Visual Studio Code (VS Code) Setup Guide

After installing Python, the next step is to install a code editor.

For this challenge, we will use **Visual Studio Code (VS Code)** because it is:

* Free
* Beginner-friendly
* Cross-platform
* Widely used in the industry
* Extensible through plugins

---

# 1. Download VS Code

Visit:

https://code.visualstudio.com

Download the version appropriate for your operating system.

Install using the default settings.

---

# 2. Launch VS Code

After installation:

* Open VS Code
* You should see the Welcome Screen

Congratulations! You now have your development environment ready.

---

# 3. Install the Python Extension

Click:

```text
Extensions
```

or press:

```text
Ctrl + Shift + X
```

Search for:

```text
Python
```

Install the extension published by:

```text
Microsoft
```

This extension provides:

* Syntax highlighting
* Code completion
* Error detection
* Debugging support
* Python interpreter integration

---

# 4. Create Your First Project Folder

Create a folder anywhere on your machine.

Example:

```text
PythonForMammals
```

Open it in VS Code:

```text
File → Open Folder
```

Select your folder.

---

# 5. Create Your First Python File

Click:

```text
New File
```

Name it:

```text
hello_mammals.py
```

---

# 6. Write Your First Program

Type:

```python
print("Hello Mammals")
```

Save the file.

Shortcut:

```text
Ctrl + S
```

---

# 7. Select Python Interpreter

Press:

```text
Ctrl + Shift + P
```

Search:

```text
Python: Select Interpreter
```

Choose the Python version you installed earlier.

Example:

```text
Python 3.14
```

---

# 8. Run Your Program

Method 1:

Click:

```text
▶ Run Python File
```

Method 2:

Open Terminal:

```text
Terminal → New Terminal
```

Run:

```bash
python hello_mammals.py
```

Expected output:

```text
Hello Mammals
```

---

# 9. Useful VS Code Shortcuts

| Shortcut         | Action          |
| ---------------- | --------------- |
| Ctrl + S         | Save file       |
| Ctrl + Shift + P | Command Palette |
| Ctrl + Shift + X | Extensions      |
| Ctrl + /         | Comment line    |
| Ctrl + `         | Open Terminal   |

---

# 10. Recommended Beginner Settings

Enable:

```text
Auto Save
```

Go to:

```text
File → Auto Save
```

This prevents losing work.

---

# Troubleshooting

## Python Interpreter Not Found

Press:

```text
Ctrl + Shift + P
```

Run:

```text
Python: Select Interpreter
```

and choose the correct version.

---

## Python Extension Missing

Install:

```text
Python (Microsoft)
```

from Extensions.

---

## Terminal Says Python Is Not Recognized

Refer to:

```text
resources/python-installation.md
```

and verify Python installation.
