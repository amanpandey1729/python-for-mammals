What you are learning today

Strings are not just “text”. In operations automation, strings are server names, log lines, file names, backup statuses, ticket IDs, alert subjects, and report rows.

string

Store text

Create text values like "SUCCESS", "server-01", and "backup.log".

len()

Measure text

Use len() to check username length, ticket length, or whether text is empty.

index

Pick characters

Use indexes like text[0] and text[-1] to read specific characters.

slice

Extract parts

Use slicing to extract environment codes, dates, file prefixes, and server numbers.

clean

Normalize messy input

Use strip(), upper(), lower(), and replace() to clean text before checking it.

search

Find useful words

Use in, startswith(), endswith(), and split() to classify logs and files.
