Day 14 • Dictionaries

Day 14: Store operational data as key-value pairs.

Today you will learn how dictionaries store structured data using meaningful keys. This is useful for server records, ticket summaries, backup reports, health checks, and future JSON-style automation.

What you will learn today

Dictionaries are one of the most important data structures for automation because they help you represent one meaningful record cleanly.

key-value

Named fields

Store related data using clear labels like server, environment, status, owner, and usage_percent.

access

Read by key

Access values using meaningful keys instead of depending on index positions.

update

Change values

Update status, usage, owner, or action values as your automation logic progresses.

loop

Print reports

Loop through items() to generate clean report-style output from structured data.

get()

Safe access

Use get() to avoid crashes when optional fields are missing from a record.

automation

Real records

Model server health, backup jobs, alerts, tickets, and inventory entries.
