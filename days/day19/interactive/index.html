What you will learn

Day 19 starts the file-handling phase. You will read text files safely and process their lines like real log and report data.

Concept 1

open()

Connect Python to a file in read mode so your script can access its content.

Concept 2

read()

Read the complete file as one string when the file is small and simple.

Concept 3

readline()

Read one line at a time when you need controlled step-by-step reading.

Concept 4

readlines()

Read all lines into a list so you can count, index, loop, and filter them.

Concept 5

with statement

Open files safely so Python automatically closes them after the block ends.

Concept 6

Line scanning

Search logs for ERROR, WARNING, FAILED, timeout, ORA-, or any keyword that matters.
