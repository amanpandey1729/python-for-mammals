What you are learning today

Each concept is tiny. Together they become the base of every useful Python script you will write later.

# comments

Leave notes for humans

Comments explain why a line exists. Python ignores them, future-you thanks you.

print()

Talk to the screen

Use print() to display messages, results and checkpoints while learning.

math

Calculate instantly

Add, subtract, multiply, divide and use powers or remainders in seconds.

variables

Name your values

A variable is a label for data, like coffee_count = 2.

assignment

Store and update

The = sign assigns a value. It does not mean “equal” like maths class.

type()

Ask Python what it sees

Use type() to check whether data is text, number, decimal or true/false.
