What you will learn today

Nested data helps you move from one record to many records, which is how most automation scripts work.

records

List of dictionaries

Store many structured records such as servers, tickets, alerts, disks, or backup jobs.

access

Read nested values

Access one record from a list, then read values using dictionary keys.

loop

Report from records

Loop through every dictionary and print useful operational summaries.

filter

Find what matters

Print only production servers, failed backups, high alerts, or warning disks.

calculate

Enrich data

Add calculated values like usage_percent, highest_usage, status, and action.

columns

Dictionary of lists

Understand column-style data and how to loop through related lists using indexes.
