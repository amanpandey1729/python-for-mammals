What you are learning today

Day 8 helped us repeat across known items. Day 9 helps us repeat until a condition changes.

while

Repeat while true

Use while when the script should continue as long as a condition remains true.

safety

Infinite loop awareness

Understand why a loop can run forever and how to update values inside the loop to avoid that.

break

Stop early

Use break when the required result is found and the remaining attempts are no longer needed.

continue

Skip one cycle

Use continue when one item or one cycle should be skipped but the loop should continue.

validate

Validation loops

Keep asking until the user enters a safe value such as DEV, TEST, PROD, or a valid percentage.

project

Controlled retry workflow

Build a small retry-style health check that stops on success and reports the final status.
