What you will learn today

Functions help you remove repeated code and build small, reliable blocks for reports, checks, and calculations.

why

Why functions?

Stop copying the same code again and again. Put repeated logic in one reusable place.

def

Defining functions

Use def to create a named block of code that can run whenever you call it.

call

Calling functions

Run a function by writing its name with parentheses, such as print_header().

input

Parameters

Pass values into a function so one function can work with many servers, tickets, or reports.

logic

Checks inside functions

Place threshold logic inside functions to keep CPU, memory, disk, and alert checks consistent.

reuse

Functions inside loops

Loop through records and call a function for each item to generate clean operational output.
