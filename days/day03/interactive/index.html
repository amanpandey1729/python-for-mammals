What you are learning today

These concepts are the starting point for real automation: ask for input, validate a value, compare it with a threshold, and print a useful result.

input()

Ask the user

Use input() when a script needs values like name, file path, server name, or usage percentage.

conversion

Convert text to numbers

User input arrives as text. Convert it with int() or float() before doing calculations.

comparison

Check a condition

Use operators like >, <, ==, and != to compare values.

if

Run code when true

An if block runs only when the condition is true.

elif

Check another possibility

elif lets your script test multiple situations without writing messy code.

else

Handle everything else

else gives your program a safe default path when no earlier condition matched.
