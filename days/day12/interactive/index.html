Day 12: Manage lists with operations.

Today you will learn how to add, insert, remove, pop, sort, reverse, and loop through list values. This is how automation scripts manage servers, alerts, files, tasks, and reports after storing them in lists.

What you are learning today

List operations help you add new values, remove completed values, reorder data, and process every item one by one.

append

Add at the end

Use append() when a new server, log file, alert, or task must be added to an existing list.

insert

Add at a position

Use insert() when a priority task or urgent alert must be placed at a specific position.

remove

Remove known values

Use remove() when a completed task or closed alert should disappear from the pending list.

pop

Pick and remove

Use pop() when you want to take one item from a queue and also store what was removed.

sort

Sort values

Use sort() to make server names, files, numbers, or report values easier to read.

loop

Process each item

Use a for loop to run the same action for every server, alert, file, or task in a list.
