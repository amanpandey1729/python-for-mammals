# 🐍 Python for Mammals

Learn Python without becoming a software developer.

A community-driven 45-Day Python Challenge designed for:

* Database Administrators
* System Administrators
* Cloud Engineers
* Support Engineers
* Middleware Teams
* Network Engineers
* Storage Teams
* Operations Professionals
* Complete Beginners

---

## Why Python for Mammals?

Most learning resources teach Python as if everyone wants to become a software engineer.

This community has a different goal.

Our mission is simple:

> Learn enough Python to automate repetitive work and save hours every week.

Examples include:

* Health check automation
* Log analysis
* CSV reporting
* Excel reporting
* Inventory collection
* Capacity tracking
* Backup verification
* Monitoring automation
* Daily operational reports

No computer science degree required.

No prior coding experience required.

Just curiosity and consistency.

---

## The 45-Day Challenge

The challenge progresses through six phases:

### Phase 1: Python Foundations

* Variables
* Data Types
* Operators
* Conditions
* Loops
* Functions

### Phase 2: Working with Data

* Strings
* Lists
* Dictionaries
* Tuples
* Sets
* File Handling

### Phase 3: Automation Essentials

* Modules
* Exception Handling
* OS Operations
* Datetime
* JSON
* CSV

### Phase 4: Professional Automation

* Pandas
* OpenPyXL
* Logging
* APIs
* Requests

### Phase 5: Operations Automation

* Health Checks
* Report Generation
* Inventory Collection
* Alert Processing

### Phase 6: Capstone Projects

* Health Check Tool
* Log Analyzer
* Backup Verification Tool
* Capacity Growth Analyzer
* Report Generator
